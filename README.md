# TicTacToeSwift

The game Tic-tac-toe in Swift, for iOS.

## Requirements
  * Xcode 6

## Running the game
  1. Open the project in Xcode
  2. Select the `TicTacToeSwift` target
  3. Select the iPhone 5 simulator
  4. Hit Run

## Running the tests
  1. Open the project in Xcode
  2. Select the `TicTacToeSwiftSpecs` target
  3. Hit Run
